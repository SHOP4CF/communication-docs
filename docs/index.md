## FIWARE data model 

[The SHOP4CF project](https://shop4cf.eu/) is about to define a data model on top of 
[FIWARE NGSI API v2](https://fiware-orion.readthedocs.io/en/stable/user/walkthrough_apiv2/index.html).  

...
